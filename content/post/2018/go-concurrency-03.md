## 并发的模式

```go
func main() {
    go println("你好, 并发!") // 干活的

    go func() { <-make(chan int) } () // 滥竽充数的, Goroutine 泄露
    go func() { for{} } () // 浪费资源的, 但不是 Goroutine 泄露
    go func() {} () // 滥竽充数的, 但不是 Goroutine 泄露

    time.Sleep(time.Second)
    println("Done")
}
```

```go
    const N = 10
    done := make(chan bool, N)

    for i := 0; i < N; i++ {
        go func(i int) {
            println(i, "你好, 并发!")
            done <- true
        }(i)
    }

    for i := 0; i < N; i++ {
        <-done
    }
}
```
